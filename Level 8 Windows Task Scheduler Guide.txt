WINDOWS TASK SCHEDULER GUIDE

Purpose:
This project can be scheduled to run automatically using Windows Task Scheduler.

Steps:

1. Open Task Scheduler.
2. Click "Create Basic Task".
3. Enter a task name.
4. Choose a trigger:
   - Daily
   - Weekly
   - At Startup
   - At Logon
5. Select "Start a Program".
6. Browse to:
   python.exe

Example:
C:\Program Files\Python314\python.exe

7. In "Add Arguments", enter the Python script path.

Example:
"E:\AI Engineer\Automation\Level 9 Ecom Folder Automation.py"

8. Click Finish.

The automation will now execute automatically based on the selected schedule.
